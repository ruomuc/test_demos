##### 基本描述

一个基于electron的excel处理程序，因为我妈不会用excel每次让我手动给她做一些筛选，一般都是金额大于多少这种比较简单筛选操作，并且记录不会很多。

##### 环境依赖

| 依赖     | 版本     |
| -------- | -------- |
| node.js  | v10.20.1 |
| electron | v9.1.2   |

##### 运行步骤

1. `npm install`
2. `npm start`

##### 打包

`npm run package`

##### 版本

| 版本号 | 功能                          |      |
| ------ | ----------------------------- | ---- |
| v1.0.0 | 数值字段的筛选，生成新的excel |      |
